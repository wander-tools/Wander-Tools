How to Use the Base64 Converter

  • Select between Text or File conversion mode
  • For text: Enter text to encode or Base64 string to decode
  • For files: Upload a file to convert to Base64 or paste Base64 to decode to file
  • Click "Encode to Base64" to convert text/file to Base64 encoding
  • Click "Decode from Base64" to convert Base64 back to original format
  • Use the "Reset All" button to clear all inputs and outputs
  • Copy the result to clipboard using the "Copy" button
  • Download the result as a file using the "Download" button
  • Clear only the output using the "Clear Output" button
  • Toggle between dark and light mode using the moon/sun button

About Base64 Encoding

  • Base64 is a encoding scheme that converts binary data to text format
  • It uses 64 different characters (A-Z, a-z, 0-9, +, /) and padding (=)
  • Commonly used for encoding data in XML, JSON, email attachments, and URLs
  • Increases data size by approximately 33% due to encoding overhead
  • Not encryption - it's encoding that can be easily decoded

Common Use Cases

  • Embedding images in HTML/CSS (data URLs)
  • Storing binary data in JSON or XML
  • Email attachments (MIME encoding)
  • Basic authentication headers
  • URL parameters for small binary data

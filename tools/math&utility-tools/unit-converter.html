How to Use the Unit Converter

  • Select the category of units you want to convert
  • Enter the value you want to convert in the "From" field
  • Select the unit to convert from in the "From" dropdown
  • Select the unit to convert to in the "To" dropdown
  • The conversion will happen automatically as you type or change units
  • Use the swap button to swap the "From" and "To" units
  • Use the "Reset" button to clear inputs and reset to default values
  • See all common conversions in the results section below
  • Toggle between dark and light mode using the moon/sun button

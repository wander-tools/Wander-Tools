How to Use the Calculator

  • Click on the numbers to enter values
  • Use the operators (+, -, ×, /) to perform calculations
  • Press the equals (=) button to see the result
  • Use the clear (C) button to reset the calculator
  • The backspace (⌫) button removes the last digit
  • The percentage (%) button calculates percentages
  • Memory functions: MC (Memory Clear), MR (Memory Recall), M+ (Memory Add), M- (Memory Subtract)
  • Scientific functions: x² (Square), √ (Square Root), sin (Sine), cos (Cosine)

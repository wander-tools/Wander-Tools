How to Use the Text Encryption Tool

  • Enter the text you want to encrypt or decrypt in the input field
  • Select an encryption method from the dropdown menu
  • Set the appropriate key for the selected encryption method
  • Click "Encrypt Text" to encrypt your text
  • Click "Decrypt Text" to decrypt encrypted text
  • Use the "Reset All" button to clear all inputs and outputs
  • Copy the result to clipboard using the "Copy" button
  • Clear only the output using the "Clear Output" button
  • Toggle between dark and light mode using the moon/sun button

Encryption Methods

  • Caesar Cipher: Shifts each letter by a fixed number of positions in the alphabet
  • Base64 Encoding: Encodes text using Base64 algorithm (not secure for encryption)
  • Reverse Text: Reverses the order of characters in the text
  • XOR Cipher: Applies XOR operation with a key for basic encryption

Security Notes

  • This tool provides basic encryption for educational purposes only
  • For truly secure encryption, use professional encryption tools
  • Never use these simple methods for sensitive data
  • The security indicator shows the relative strength of each method

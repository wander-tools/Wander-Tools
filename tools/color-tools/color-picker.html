How to Use the Color Picker

  • Use the color wheel to select a color visually
  • Adjust the RGB sliders to fine-tune your color
  • Enter a HEX code directly in the input field
  • Use the "Pick Color from Screen" button to select any color from your screen
  • Copy color values in different formats (HEX, RGB, HSL)
  • Add colors to your palette for future reference
  • Test contrast between your color and white/black text
  • Use the "Reset" button to restore default settings
  • Toggle between dark and light mode using the moon/sun button

Color Formats Explained

  • HEX: Hexadecimal representation (e.g., #6A11CB)
  • RGB: Red, Green, Blue values (e.g., rgb(106, 17, 203))
  • HSL: Hue, Saturation, Lightness (e.g., hsl(269, 85%, 43%))

Color Theory Basics

  • Hue: The color itself (red, blue, green, etc.)
  • Saturation: The intensity or purity of the color
  • Lightness: How light or dark the color is
  • Contrast: The difference in luminance between foreground and background

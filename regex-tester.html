    How to Use the Regex Tester

    • Enter your regular expression in the "Regular Expression" field (with or without slashes)
    • Select the appropriate flags for your regex (global, case insensitive, etc.)
    • Enter the text you want to test in the "Test String" field
    • Click "Test Regex" to see the results
    • Matches will be highlighted in the results area and listed below
    • Use the "Reset All" button to clear all inputs and results
    • Toggle between dark and light mode using the moon/sun button

    Regex Flags

    • g (Global): Find all matches rather than stopping after the first match
    • i (Ignore Case): Case-insensitive matching
    • m (Multiline): Treat beginning and end characters (^ and $) as working across multiple lines
    • u (Unicode): Treat pattern as a sequence of Unicode code points
    • y (Sticky): Matches only from the index indicated by the lastIndex property of the regex

    Common Regex Patterns

    Email: /^[^\s@]+@[^\s@]+\.[^\s@]+$/
    URL: /https?:\/\/[^\s/$.?#].[^\s]*/
    Phone: /(\+\d{1,3}\s?)?(\(\d{1,4}\)\s?)?\d{1,4}[\s-]?\d{1,4}[\s-]?\d{1,9}/
    Date (YYYY-MM-DD): /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12][0-9]|3[01])$/

    Tips & Tricks

    • Use parentheses () to create capturing groups
    • Use square brackets [] to define character classes
    • Use curly braces {} to specify repetition
    • Escape special characters with a backslash (e.g., \., \+, \*)
    • Test with various inputs to ensure your regex works correctly in all cases

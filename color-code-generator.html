How to Use the Color Code Converter

  • Select the input format (HEX, RGB, or HSL)
  • Enter your color code in the input field
  • Click "Convert Color" to convert to other formats
  • Use the eye dropper to select a color from the screen
  • Copy individual color codes or all formats at once
  • Generate random colors for inspiration
  • Explore color schemes (complementary, analogous, triadic)
  • Check color luminance and contrast ratios
  • Use the "Reset" button to clear all inputs
  • Toggle between dark and light mode using the moon/sun button

Color Formats Explained

  • HEX: Hexadecimal representation (e.g., #6A11CB)
  • RGB: Red, Green, Blue values (e.g., rgb(106, 17, 203))
  • HSL: Hue, Saturation, Lightness (e.g., hsl(269, 85%, 43%))

Color Schemes

  • Complementary: Colors opposite each other on the color wheel
  • Analogous: Colors next to each other on the color wheel
  • Triadic: Three colors evenly spaced on the color wheel

Accessibility Guidelines

  • Text should have a contrast ratio of at least 4.5:1 for normal text
  • Large text should have a contrast ratio of at least 3:1
  • Luminance values help determine color brightness

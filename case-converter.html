Converted Text

How to Use the Case Converter

  • Type or paste your text into the input text area
  • Select the desired case conversion option
  • Click the "Convert Text" button to convert your text
  • Use the "Reset" button to clear all inputs and outputs
  • Copy the converted text to clipboard using the "Copy" button
  • Clear only the output using the "Clear Output" button
  • Toggle between dark and light mode using the moon/sun button

Case Conversion Types

  • UPPER CASE: Converts all text to uppercase letters
  • lower case: Converts all text to lowercase letters
  • Title Case: Capitalizes The First Letter Of Each Word
  • Sentence case: Capitalizes the first letter of each sentence
  • camelCase: capitalizesTheFirstLetterOfEachWordExceptTheFirst
  • PascalCase: CapitalizesTheFirstLetterOfEachWord
  • snake_case: separates_words_with_underscores
  • kebab-case: separates-words-with-hyphens
